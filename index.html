¿Qué vas a encontrar aquí?

Los ejercicios hechos cada día en clase, con sus fallos, ¿aciertos? y cosas random.

Dia 1

El primer día es algo light para entrar en calor. Un framework de css que añade los filtros de instagram.

Dia 2

Ahí hay un slider. Los botones no se ven por el color del fondo, pero se puede ver en funcionamiento arrastrando las imágenes o con la reporducción automática. Por cierto el css mío afecta al indice del slider (lol). También hay Buttons de muchas fomras y colores, Accordion y avisos de alert con SweetAlert2.

Dia 3

En el día 3 hay Cookies! seguido de Awesomplate para autocompletar campos de formularios, Selectize para esas busquedas chulas con tags y dos editores de contenido, Trumbowyg y Quill.

Dia 5

Aviso a tripulantes, documento muy pesado. El framework Graphviz, para hacer grafos, pesa ¡10Mb! asi que igual te da tiempo a darte una , revisar o, como yo acabo de hacer, mirar iconos en Font Awesome.

Tambien hay cosas mas ligeritas como Chartist para las gráficas, Js Sequence Diagrams para unos diagramas con JavaScript, TableSort para poder ordenar tablas, Flip para tarjetas de presentación que se dan la vueltay por último Panorama que nos da una vista panorámica chula para nuestras fotos (Aunque yo usé una de prueba).

Dia 5

Presentación en 3D. Hecho con Impress para dejarte impress-ionado.
